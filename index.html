User Study

Context

This user study collects data about the reaction time of humans based on different stimuli.

There are three different type of stimuli:

  • Unary Stimulus: A circle slowly appearing on a random position on the screen. The user has to react as soon as he recognizes the circle.
  • Binary Stimulus: Circles and Triangles appear on the screen. The user has to react as soon as he recognizes a triangle.
  • Complex Stimulus: Images of (Italian, Chinese or Mexican) food is shown. The user has to press different buttons based on the origin country of the food.

This study was designed by Jonas Unterweger, Paula Kostron and Florian Schad.
